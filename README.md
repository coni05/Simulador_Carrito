# 🛒 Simulador de Carrito de Compras

Aplicación web de simulación de carrito de compras desarrollada con HTML, CSS y JavaScript puro, aplicando programación funcional y pruebas unitarias con medición de cobertura de código.

## 📋 Características

- ✅ Catálogo de productos en cards
- ✅ Agregar productos al carrito
- ✅ Eliminar productos del carrito
- ✅ Incrementar/decrementar cantidades
- ✅ Cálculo de subtotal, impuestos y total
- ✅ Persistencia en LocalStorage
- ✅ Diseño responsive y atractivo
- ✅ Pruebas unitarias con >60% de cobertura

## 🚀 Requisitos Previos

- Node.js (v14 o superior)
- npm (v6 o superior)
- Navegador web moderno

## 📦 Instalación

1. Clona o descarga el repositorio
2. Navega al directorio del proyecto
3. Instala las dependencias:

```bash
# Instalar dependencias
npm install

# Ejecutar la aplicación
npm start

# Correr tests
npm test

# Ver cobertura
npm run test:coverage
